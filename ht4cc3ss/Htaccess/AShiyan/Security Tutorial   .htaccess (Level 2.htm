    با سلام

    اینم تعدادی .htaccess که خیلی به امنیت سایت کمک میکنه



    برای جلوگیری از اجرای (.pl .cgi .php .py ) (خودتون به صورت دستی میتونید فرمتهای دیگه هم اضافه کنید ، مثل php4 , php5 و...)

    کد PHP:
    RemoveHandler cgi-****** .pl .cgi .php .py 
    AddType text    /plain .pl .cgi .php .py 

    Anti Dos :

    کد PHP:
    # drop Range header when more than 5 ranges.
    # CVE-2011-3192
    SetEnvIf Range (,.*?){5,} bad-range=1
    RequestHeader     unset Range env=bad-range
    # optional logging.
    #CustomLog insert-path-and-name-of-log common env=bad-range 

    پیچیده کردن CSRF :

    کد PHP:
    Header set X-Frame-Options DENY 

    جالب :انتقال سایر فایل های index به صفحه اصلی

    با این Htaccess میتوانید صفحه Index پیش فرض سایت رو تغییر بدید و سایر فایل ها را به صفحه اصلی خود redirect کنید. به این ترتیب در صورت ایجاد سایر فایل های index این فایل ها پیش فرض نیستند و محتوا صفحه اصلی شما تغییر نمی کنند و همچنین در صورت باز شدن مستقیم این فایل اجرا نمی شود و به صفحه اصلیتون منتقل می شود :

    کد PHP:
    RedirectMatch 301 /index.html http://www.yoursite.com/page.php 
    مثلا در اینجا index.html رو به www.yoursite.com/page.php تغیید میده
